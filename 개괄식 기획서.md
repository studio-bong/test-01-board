# 🤖 Antigravity Agent 지시용 핵심 기획서

## [프로젝트 개요]

* **목표:** Glitch 배포용 Node.js Express 기반 간단한 온라인 게시판 구현
* **스택:** Vanilla JS (Frontend) + Node.js/Express (Backend) + db.json (File DB)
* **진행 모드:** Review-driven (각 단계 실행 전 구현 계획 승인 필수)

---

## [Task List & 지시 가이드]

AI 에이전트에게 아래 순서대로 **Task List와 Implementation Plan**을 수립하도록 요청하세요.

1. **환경 초기화 및 데이터 구조 설계:** Task 1.
`package.json` 생성 및 Express 라이브러리 추가, 빈 게시글 배열(`{ "posts": [] }`)을 가진 `db.json` 파일 생성 계획 수립.


2. **백엔드 API 구현 (server.js):** Task 2.
글 쓰기(`POST /api/posts`) 및 글 읽기(`GET /api/posts`) API 라우트 구현 계획 작성. 내장 `fs` 모듈을 사용해 `db.json`에 실시간으로 데이터를 읽고 쓰도록 설정.


3. **프론트엔드 UI 및 비동기 통신 구현:** Task 3.
`index.html` 및 `script.js` 생성 계획 수립. Fetch API를 사용하여 화면 전환 없이 백엔드 API와 데이터를 주고받는 로직 연결.


4. **Browser Subagent 검증 및 코드 인도:** Task 4.
Antigravity 내장 Browser를 띄워 실시간 데이터 누적 기능 테스트. 최종 무결성이 확인되면 Glitch에 붙여넣을 수 있도록 파일별 완성 코드 정리(Walkthrough 소스 제공).
