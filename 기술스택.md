## 🛠️ 온라인 게시판 개발 기술 스택 (최종본)

### 1. 프론트엔드 (내 컴퓨터 / 스마트폰)

* **HTML5 / CSS3 / JavaScript (Vanilla JS)**
* 사용자가 접속하는 게시판 화면입니다.
* 내 컴퓨터에서 열어도 되고, 스마트폰으로 열어도 인터넷을 통해 서버와 통신합니다.



### 2. 백엔드 (온라인 클라우드 서버)

* **Node.js + Express**
* 24시간 꺼지지 않는 인터넷 환경(예: Glitch 등) 위에서 구동됩니다.
* 게시글을 저장하고 불러오는 핵심 API(우편물 수거함) 역할을 합니다.



### 3. 데이터베이스 (온라인 파일 저장소)

* **db.json (JSON 텍스트 파일)**
* 온라인 서버 컴퓨터 내부에 저장되는 장부 파일입니다.
* 복잡한 데이터베이스 프로그램 없이 텍스트 형태로 데이터가 실시간 누적됩니다.



### 4. 개발 및 인프라 환경

* **인프라**: **Glitch** (또는 Replit/Render 등 초보자용 클라우드 서비스)
* 별도의 로컬 프로그램 설치 없이 웹 브라우저만으로 온라인 서버를 구축하고 실시간 배포 주소(`https://...`)를 획득합니다.
